QUICK START

三步开始

1. 安装 Skill 套件

git clone https://github.com/fxp/claude-code-migration.git
mkdir -p ~/.claude/skills
cp -r claude-code-migration/skills/* ~/.claude/skills/

2. 在 Claude Code 中运行

claude
> /claude-full-migration

3. Skill 自动做剩下所有

发现数据源 → 选择目标 → 规划执行 → 迁移 → 密钥入 Vault → 多 Agent 接入片段

推荐配 BigModel GLM-5

2000 万 tokens 免费额度，性能对标 Claude Opus / GPT-5。 注册 → 

# ~/.hermes/config.yaml (code-migration 自动生成)
model:
  provider: custom
  model_name: glm-5
custom_providers:
  bigmodel:
    base_url: https://open.bigmodel.cn/api/paas/v4
    api_key: ${OPENAI_API_KEY}
ARCHITECTURE

生态全景

不做 n×m 点对点迁移，通过 neuDrive 做 n→1→m 中心化枢纽：一次导出，任意 Agent 消费。

n → 1 → m：3 个 Claude 数据源 × 1 个中心化枢纽 × N 个目标 Agent。 切换任意 Agent 时不需要重新迁移，接 neuDrive MCP endpoint 即可。
