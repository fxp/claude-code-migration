QUICK START

两种使用方式

Python 包可脚本化、可测试、可 CI；Skills 版本无需 Python 环境。

方式 A · Python 包（推荐）

git clone https://github.com/fxp/claude-code-migration
cd claude-code-migration
python3 -m venv .venv && source .venv/bin/activate
pip install -e .

扫描一个项目：

ccm scan --project /path/to/your-project

一次迁到 4 个目标（默认 staging，不动真实项目）：

ccm migrate --project /path/to/your-project \
            --target hermes,opencode,cursor,windsurf \
            --out ./ccm-output

可选：加上 Claude.ai 官方导出 ZIP（Chat / Cowork）：

ccm migrate --project /path/to/your-project \
            --cowork-zip ~/Downloads/claude-data-export.zip \
            --target hermes,opencode --out ./ccm-output

推送到 neuDrive Hub：

ccm push-hub --scan ./ccm-output/scan.json \
             --token $NEUDRIVE_TOKEN \
             --api-base https://www.neudrive.ai

方式 B · Claude Code Skills（无 Python 环境）

git clone https://github.com/fxp/claude-code-migration
mkdir -p ~/.claude/skills
cp -r claude-code-migration/skills/* ~/.claude/skills/

然后在 Claude Code 里：

claude
> /claude-full-migration

推荐配 BigModel GLM-5

2000 万 tokens 免费额度，性能对标 Claude Opus / GPT-5。 注册 → Adapter 自动为每个目标生成对应 provider 配置（Hermes YAML / OpenCode JSON）。
